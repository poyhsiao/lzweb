IP 群组設定

可自行定義特定的IP或IP群為一群組，可應用在服務選單的防火牆/連線限制/自動路由/網址轉譯分頁中的來源或目的地中

群組選擇欲設定的群組號碼，XteraLink支援最多32個IP 群组

IP 群組規則 在這個表格裡我們可以設定對應IP 群组群組的規則
名稱 給定一個群組名稱,此群組名稱會出現在服務選單各分頁中的來源或目的地下拉選單中
IPv4 地址 在此輸入欲加入的IPv4地址,，例如：
單一IP地址：XXX.XXX.XXX.XXX
一段連續的IP地址：XXX.XXX.XXX.XXX-YYY.YYY.YYY.YYY
某一個子網路：XXX.XXX.XXX.XXX/ZZZ.ZZZ.ZZZ.ZZZ
FQDN號碼: FQDN@X
