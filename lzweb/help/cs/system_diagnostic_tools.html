网络诊断工具

网络诊断工具 在这个表格里我们可以使用ARP 重整、IP 冲突测试工具
ARP 重整 ARP重整强迫XteraLink周遭的主机或设备更新ARP table
点选[重整]，系统会送出arp reply封包以更新周遭的主机或设备的ARP table，通常用在当部分DMZ的设备在第一次安装XteraLink后不能正确的与 internet 联机时才须使用
IP 冲突测试 IP 冲突测试寻找目前网络上是否有主机的位置与XteraLink的网络设定分页中的设定有相违背的状况，系统会侦测到以下的冲突状况发生:
1.在隔离区(DMZ)中发现有与设定在广域网络端(WAN)相同的IP地址
2.在广域网络端(WAN)中发现有与设定在隔离区(DMZ)相同的IP地址
3.在广域网络端(WAN)、局域网络(LAN)或隔离区(DMZ)中发现有与设定给XteraLink本机相同的IP地址

Ping 与 路由跟踪 在这个表格里我们可以在指定的接口上使用Ping或路由跟踪工具
目标 IP 输入目标IP或FQDN
注：若使用FQDN时，须web UI上的[系统]->[域名称服务器]->[域名称服务器]中先行指定域名称服务器
界面 选择使用的接口，例如：广域网络、局域网络、隔离区
Ping 显示Ping从指定接口到目标 IP的结果
跟踪 显示封包从指定接口到目标 IP中间经过的路由
停止 用来中断Ping或路由跟踪程序
