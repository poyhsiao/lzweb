Diagnostic Tools

Diagnostic Tools This table allows you to use ARP Enforcement and IP Conflict Test tools.
ARP Enforcement This tool is used to enforce the updates of ARP table on hosts or devices connected to XteraLink.
Click Enforce, and system will send out arp reply packets to update the ARP table of hosts or devices. This is usually used when some devices at DMZ site fail to connect to Internet normally after installing XteraLink for the first time.
IP Conflict Test This tool is used to detect whether any host's IP address is in conflict with any IP address defined in XteraLink's configuration pages. Possible cases of conflict might include:
1. An IP address at DMZ site is found duplicated with that defined in WAN.
2. An IP address at WAN site is found duplicated with that defined in DMZ.
3. An IP address at WAN, LAN or DMZ site is found duplicated with the hosting IP address of XteraLink.

Ping & Trace Route This table allows you to use Ping or Trace Route tool on specified interface.
Target IP Enter a target IP address or FQDN number.
Note: In order to use an FQDN number, a domain name must be specified on the page of: [System > DNS] first.
Interface Select an interface: WAN, LAN or DMZ.
Ping This shows a Ping result between selected interface and target IP.
Trace Route This shows a result of routes traced between selected interface and target IP.
Stop This is used to stop Ping or Trace Route diagnosis.
