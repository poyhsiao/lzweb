Service Group

TCP/UDP ports or protocol numbers can be defined as a group. Ports defined for specific network application or server can be applied to configurations in many Service pages (e.g. Virtual Server/Firewall/Connection Limit/Auto Routing/NAT).

GroupSelect a group number to be configured. XteraLink supports up to 32 IP groups.

Service Group Rule This table allows you to set up a group rule for the selected service group.
Label This is the name of the service group. This service group name will appear as a service option in the drop-down menu in many Service configuration pages.
Service Enter an IPv4 address here to be included in the group. e.g.
A single protocol: proto@XX
A single port: TCP@XXX, UDP@XXX
a port range: TCP@XXX-YYY, UDP@XXX-YYY
