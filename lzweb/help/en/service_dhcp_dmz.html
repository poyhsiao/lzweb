DHCP (DMZ)

IPv4 DHCP (DMZ) Setting This table allows you to configure DNS Server, Gateway, Netmask, DHCP Range and Static Mapping.
DNS Server Enter the IP address of the DNS server assigned for hosts in DMZ. e.g. 192.168.0.254
Gateway Enter a gateway IP for hosts in DMZ. e.g. 192.168.0.1
Note: The IP address of gateway must be one of those defined in [System/Network Setting/DMZ Setting/IPv4 Basic Subnet/IP Address/Range].
Netmask Enter a netmask for hosts added in DMZ. e.g. 255.255.255.0
Note: The netmask configured must be included in existing DMZ's netmask.
DHCP Range Enter a range of dynamic IP addresses assigned for hosts in DMZ. e.g.
Starting Address: 192.168.0.10
Ending Address: 192.168.0.20
Note: The Starting and Ending addresses must be included in existing DMZ's netmask.
Static Mapping Enter a static IP address and corresponding MAC address of the host in DMZ. e.g.
MAC Address: xx:xx:xx:xx:xx:xx
IP Address: 192.168.0.100
Note: The IP address configured must be included in existing DMZ's netmask.
