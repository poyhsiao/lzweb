IP Group

Different IP addresses can be put together as a group. IP groups defined here can be applied to the configurations of Source or Destination IP in many Service pages (e.g. Firewall/Connection Limit/Auto Routing/NAT).

GroupSelect a group number to be configured. XteraLink supports up to 32 IP groups.

IP Group Rule This table allows you to set up a group rule for the selected group.
Label This is the name of the group. This group name will appear as a drop-down menu option (e.g. Source or Destination IP) in many Service configuration pages.
IPv4 Address Enter an IPv4 address here to be included in the group. e.g.
A single IP: XXX.XXX.XXX.XXX
An IP range: XXX.XXX.XXX.XXX-YYY.YYY.YYY.YYY
An IP subnet/mask: XXX.XXX.XXX.XXX/ZZZ.ZZZ.ZZZ.ZZZ
FQDN number: FQDN@X
