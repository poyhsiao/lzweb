ARP Table

Fixed Entries Setting This table allows you to add or modify the records in system's ARP table.
Interface Select an interface to add to the entry. e.g. LAN, WAN, DMZ, etc.
IP Address Enter an IP address. e.g. x.x.x.x
MAC Address Enter a corresponding MAC address. e.g. xx:xx:xx:xx:xx:xx

Dynamic Entries Setting This table displays the records of all ARP tables in system.
Interface The interfaces recorded in current system's ARP table.
IP Address The IP addresses recorded in current system's ARP table.
MAC Address The MAC addresses recorded in current system's ARP table.
Fix This option is unchecked (disabled) by default.
When checked (enabled) and after configuration applied, this entry will be added to Fixed Entries Setting.
