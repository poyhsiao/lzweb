WAN Setting

WANSelect a WAN number from the drop-down menu.

EthernetThis table includes configurations of the WAN's corresponding network card, such as Clone MAC, MTU and Speed/Duplex.
Clone MACSet up a MAC address for the selected WAN.
MTUAdministrators are allowed to define the size of every packet here. Large data will be transmitted in several packets.
Speed/DuplexSelect a speed/duplex transmission option from the drop-down menu for the selected WAN.

Basic SettingThis table allows you to make configurations of the basic subnet.
EnableWhen disabled (unchecked), the selected WAN will be unavailable.
When enabled (checked), configurations of the selected WAN will take effect after applied.
LabelEnter a name for the selected WAN.
WAN TypeSelect a protocol type from the drop-down menu: Static, PPPoE and DHCP.
StaticWhen selected, an IPv4 Static Type table below will appear for configuration.
IPv4 Static TypeThis table includes configurations of Gateway, Netmask, and IP address/Range.
Gateway The IP address of your ISP, i.e. the default gateway. e.g. 211.21.40.254
NetmaskThe allocated netmask from ISP. e.g. 255.255.255.0
IP Address/RangeThe allocated IP address or IP range from ISP.
IP Address: e.g. 211.21.40.32
IP Range: e.g. 211.21.40.32-211.21.40.42
PPPoEWhen selected, a PPPoE Type table will appear for configuration.
PPPoE TypeThis table includes configurations of User Name, Password, Service Name, IPv4 Address, and Daily Redial.
User NameThe username required for dial-up connection to ISP.
PasswordThe password required for dial-up connection to ISP.
Service NameThe name of the service required for dial-up connection.
IPv4 AddressThe IP address requested from ISP.
Daily RedialBecause some ISPs tend to automatically disconnect PPPoE lines regularly at certain time, XteraLink will re-establish connection for every disconnected PPPoE line simultaneously when detected.
In order to avoid multiple line redial, you may set up different redial time here for different WAN lines.
After setting redial time (HH:MM), the system will redial to establish connection for the PPPoE line at the configured time.
DHCPWhen selected, IPs are allocated dynamically for connections.
DownstreamThe download rate of this line, e.g. 512 (Kbps)
UpstreamThe upload rate of this line, e.g. 512 (Kbps)

USB Modem StatusThis table allows you to view details of Status, Brand/Model, Operator, Mode and Quality.
StatusThis indicates the status of the USB modem, e.g.
Unknown: USB modem not supported or Not connected to USB modem
Plug: Connected to USB modem
PIN Error: Incorrect PIN entry
Dial-in Failed: Unable to dial in
Dial-in OK: Dialed in successfully
Brand/modelThis indicates the brand/model of the USB modem in use. e.g. huawei/E177
OperatorThis indicates the current operator of the USB modem. e.g. Chunghwa Telecom
ModeThis indicates how USB modem is connected currently. e.g.
via WCDMA (Wide band Code Division Multiple Access)
via HSPA (High Speed Packet Access)
QualityThis illustrates the signal quality of the USB modem. e.g.
No signal
Low reception
Moderate reception
Strong reception
Best reception

3G/3.5G Dial-up SettingThis table allows you to configure the PIN, dial number, APN, user name, password and MTU.
PINThis indicates the PIN code (Personal identification number) of the SIM card. e.g. 0000
APNThis indicates the APN (Access Point Name) used. e.g. internet
Dial numberThis indicates the number to dial up. e.g. *99#
User NameThis indicates the user name used for dial-up account. e.g. xxxx
PasswordThis indicates the password used for dial-up account. e.g. xxxx
MTUThis indicates the MTU (Maximum transmission unit) after dialed in. e.g. 1500

IPv4 Public IP PassthroughThis table allows you to add an IP address/range of the subnet in the selected WAN, while the IP addresses of hosts behind DMZ can be set up for IP Passthrough.
IP Address/RangeThe IP address/range of hosts behind DMZ
IP address: e.g. 211.21.40.33
IP Range: e.g. 211.21.40.43-211.21.40.53

DMZ Setting

EthernetThis table includes configurations of the DMZ's corresponding network card, such as Clone MAC, MTU and Speed/Duplex.
Clone MACSet up a MAC address for the DMZ.
MTUAdministrators are allowed to define the size of every packet here. Large data will be transmitted in several packets.
Speed/DuplexSelect a speed/duplex transmission option from the drop-down menu for the DMZ.

IPv4 Basic SubnetThis table includes configurations of IP Address/Range and Netmask.
IP Address/RangeThe IP address/range of XteraLink in DMZ, as well as the default gateway in DMZ.
IP Address: e.g. 192.168.10.1
IP Range: e.g. 192.168.10.1-192.168.10.10
NetmaskThe netmask of XteraLink in DMZ, e.g. 255.255.255.0

IPv4 Static Routing SubnetThis table includes configurations of Subnet/Mask and Gateway.
Subnet/MaskA private network segment in DMZ.
GatewayThe gateway where static routing is used in the private network segment.
