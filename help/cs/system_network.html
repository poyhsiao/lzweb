广域网络设定

广域网络下拉选单选择欲设定的广域网络号码

以太网络在这个表格里可以设定此广域网络对应网卡的Clone MAC、MTU、传输模式
Clone MACXteraLink 可让您设定这种广域网络网卡的 MAC 地址
MTU管理员可以在此处设定每个数据包的大小，若一段数据过大将被分为多个包传送
传输模式选择此广域网络接口上的传输模式

基本设定在这个表格里可以设定启用、名称、联机模式、下载带宽、上传带宽
启用当不启用(Disable)此设定时，此广域网络无法使用
当启用(Enable)此设定时，此广域网络会依照使用者按下套用设定时的设定启用
名称使用者自行设定的广域网络命名
联机模式在这个下拉选单里可以设定三种广域网络联机模式，包含固定、PPPoE、DHCP动态配址
固定联机模式选择固定，会出现IPv4 静态模式表格
IPv4 静态模式表格在这个表格里可以填入网关器、网络掩码、IP 地址/范围
网关器 为ISP端的IP地址，即预设网关器地址，例如：211.21.40.254
网络掩码ISP配发的网络屏蔽，例如：255.255.255.0
IP 地址/范围ISP配发的IP地址或范围
地址，例如：211.21.40.32
范围，例如：211.21.40.32-211.21.40.42
PPPoE联机模式选择PPPoe，会出现PPPoE 模式表格
PPPoE 模式在这个表格里可以填使用者名称、密码、服务名称、IPv4 地址、每日重拨时间
使用者名称设定拨接上ISP时所需要的使用者名称
密码设定拨接上ISP时所需要的密码
服务名称设定拨号时ISP所需要的服务名称
IPv4 地址填入向ISP申请的IP地址
每日重拨时间某些ISP在固定时间段后会自动切断PPPoE联机，因此当XteraLink侦测到线路断线后会同时替每条断线的PPPoE线路重拨
为避免同时间多条线路回放的情况，可在此针对不同WAN线路设定不同重拨时间，将这些PPPoE线路重拨时间错开
设定HH:MM回放时间后，系统会每天固定在此时间点为该PPPoE线路进行回放
DHCP动态配址使用DHCP自动联机
下行流量此线路提供的下载速率，例如512 (Kbps)
上行流量此线路提供的上传速率，例如512 (Kbps)

USB 调制解调器 状态在这个表格里可以察看状态、品牌/型号、电信公司、联机模式、讯号质量
状态显示此USB调制解调器的状态，例如：
Unknown: 不支持的USB设备或USB设备未接上
Plug：USB调制解调器已接上
PIN Error：PIN输入错误
Dial-in Failed：拨号失败
Dial-in OK：拨号成功
品牌/型号显示此USB调制解调器的品牌/型号，例如：huawei/E177
电信公司显示此USB调制解调器目前的电信公司，例如：Chunghwa Telecom
联机模式显示此USB调制解调器目前的联机模式，例如：
WCDMA (Wide band Code Division Multiple Access)
HSPA (High Speed Packet Access)
讯号质量以图形显示此USB调制解调器目前的讯号质量，例如：
无讯号
讯号微弱
讯号中等
讯号强
讯号最强

3G/3.5G 拨号 设定在这个表格里可以设定PIN、拨接号码、APN、使用者名称、密码、MTU
PIN输入SIM卡的PIN(Personal identification number)，例如：0000
APN输入拨号使用的APN(Access Point Name)，例如：internet
拨接号码输入拨号使用的拨接号码，例如：*99#
使用者名称输入拨号使用的使用者名称，例如：xxxx
密码输入拨号使用的密码，例如：xxxx
MTU输入拨号完成后设备的MTU(Maximum transmission unit)，例如：1500

IPv4 公开 IP 穿透在这个表格里可以增加此广域网络子网络中的IP地址或范围，在隔离区后的主机可以设定其IP地址做网络透通
IP 地址/范围隔离区后的主机的IP地址或范围
地址，例如：211.21.40.33
范围，例如：211.21.40.43-211.21.40.53

隔离区设定

以太网络在这个表格里可以设定此隔离区对应网卡的Clone MAC、MTU、传输模式
Clone MACXteraLink 可让您设定隔离区网卡的 MAC 地址
MTU管理员可以在此处设定每个数据包的大小，若一段数据过大将被分为多个包传送
传输模式选择此隔离区接口上的传输模式

IPv4 子网络设定在这个表格里可以填入IP 地址/范围、网络掩码
IP 地址/范围设定XteraLink本身位于隔离区接口的IP地址或范围，其用途为当成位于隔离区中客户的预设网关器地址
IP 地址，例如：192.168.10.1
范围，例如：192.168.10.1-192.168.10.10
网络掩码设定XteraLink本身位于隔离区接口的网络掩码，例如：255.255.255.0

IPv4 静态路由子网络在这个表格里可以填入子网络/网络屏蔽、网关器
子网络/网络屏蔽为隔离区中某一段私有网段
网关器为此私有网段需做静态路由之网关器地址
