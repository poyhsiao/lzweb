IP 群组设定

可自行定义特定的IP或IP群为一群组，可应用在服务选单的防火墙/连接限制/自动路由/网址转译分页中的来源或目的地中

群组选择欲设定的群组号码，XteraLink支持最多32个IP 群组

IP 群组规则 在这个表格里我们可以设定对应IP 群组群组的规则
名称 给定一个群组名称,此群组名称会出现在服务选单各分页中的来源或目的地下拉选单中
IPv4 地址 在此输入欲加入的IPv4地址,，例如：
单一IP地址：XXX.XXX.XXX.XXX
一段连续的IP地址：XXX.XXX.XXX.XXX-YYY.YYY.YYY.YYY
某一个子网络：XXX.XXX.XXX.XXX/ZZZ.ZZZ.ZZZ.ZZZ
FQDN号码: FQDN@X
