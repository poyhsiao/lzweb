廣域網路設定

廣域網路下拉選單選擇欲設定的廣域網路號碼

乙太網路在這個表格裡可以設定此廣域網路對應網卡的Clone MAC、MTU、傳輸模式
Clone MACXteraLink 可讓您設定這種廣域網路網卡的 MAC 地址
MTU管理員可以在此處設定每個數據包的大小，若一段數據過大將被分為多個包傳送
傳輸模式選擇此廣域網路介面上的傳輸模式

基本設定在這個表格裡可以設定啟用、名稱、連線模式、下載頻寬、上傳頻寬
啟用當不啟用(Disable)此設定時，此廣域網路無法使用
當啟用(Enable)此設定時，此廣域網路會依照使用者按下套用設定時的設定啟用
名稱使用者自行設定的廣域網路命名
連線模式在這個下拉選單裡可以設定三種廣域網路連線模式，包含固定、PPPoE、DHCP動態配址
固定連線模式選擇固定，會出現IPv4 靜態模式表格
IPv4 靜態模式表格在這個表格裡可以填入閘道器、網路遮罩、IP 地址/範圍
閘道器 為ISP端的IP地址，即預設閘道器地址，例如：211.21.40.254
網路遮罩ISP配發的網路遮罩，例如：255.255.255.0
IP 地址/範圍ISP配發的IP地址或範圍
地址，例如：211.21.40.32
範圍，例如：211.21.40.32-211.21.40.42
PPPoE連線模式選擇PPPoe，會出現PPPoE 模式表格
PPPoE 模式在這個表格裡可以填使用者名稱、密碼、服務名稱、IPv4 地址、每日重播時間
使用者名稱設定撥接上ISP時所需要的使用者名稱
密碼設定撥接上ISP時所需要的密碼
服務名稱設定撥號時ISP所需要的服務名稱
IPv4 地址填入向ISP申請的IP地址
每日重播時間某些ISP在固定時間段後會自動切斷PPPoE連線，因此當XteraLink偵測到線路斷線後會同時替每條斷線的PPPoE線路重撥
為避免同時間多條線路重播的情況，可在此針對不同WAN線路設定不同重撥時間，將這些PPPoE線路重撥時間錯開
設定HH:MM重播時間後，系統會每天固定在此時間點為該PPPoE線路進行重播
DHCP動態配址使用DHCP自動連線
下載頻寬此線路提供的下載速率，例如512 (Kbps)
上傳頻寬此線路提供的上傳速率，例如512 (Kbps)

USB 數據機 狀態在這個表格裡可以察看狀態、品牌/型號、電信公司、連線模式、訊號品質
狀態顯示此USB數據機的狀態，例如：
Unknown: 不支援的USB設備或USB設備未接上
Plug：USB數據機已接上
PIN Error：PIN輸入錯誤
Dial-in Failed：撥號失敗
Dial-in OK：撥號成功
品牌/型號顯示此USB數據機的品牌/型號，例如：huawei/E177
電信公司顯示此USB數據機目前的電信公司，例如：Chunghwa Telecom
連線模式顯示此USB數據機目前的連線模式，例如：
WCDMA (Wide band Code Division Multiple Access)
HSPA (High Speed Packet Access)
訊號品質以圖形顯示此USB數據機目前的訊號品質，例如：
無訊號
訊號微弱
訊號中等
訊號強
訊號最強

3G/3.5G 撥號 設定在這個表格裡可以設定PIN、撥接號碼、APN、使用者名稱、密碼、MTU
PIN輸入SIM卡的PIN(Personal identification number)，例如：0000
APN輸入撥號使用的APN(Access Point Name)，例如：internet
撥接號碼輸入撥號使用的撥接號碼，例如：*99#
使用者名稱輸入撥號使用的使用者名稱，例如：xxxx
密碼輸入撥號使用的密碼，例如：xxxx
MTU輸入撥號完成後設備的MTU(Maximum transmission unit)，例如：1500

IPv4 公開 IP 穿透在這個表格裡可以增加此廣域網路子網路中的IP地址或範圍，在隔離區後的主機可以設定其IP地址做網路透通
IP 地址/範圍隔離區後的主機的IP地址或範圍
地址，例如：211.21.40.33
範圍，例如：211.21.40.43-211.21.40.53

隔離區設定

乙太網路在這個表格裡可以設定此隔離區對應網卡的Clone MAC、MTU、傳輸模式
Clone MACXteraLink 可讓您設定隔離區網卡的 MAC 地址
MTU管理員可以在此處設定每個數據包的大小，若一段數據過大將被分為多個包傳送
傳輸模式選擇此隔離區介面上的傳輸模式

IPv4 子網路設定在這個表格裡可以填入IP 地址/範圍、網路遮罩
IP 地址/範圍設定XteraLink本身位於隔離區介面的IP地址或範圍，其用途為當成位於隔離區中客戶的預設閘道器地址
IP 地址，例如：192.168.10.1
範圍，例如：192.168.10.1-192.168.10.10
網路遮罩設定XteraLink本身位於隔離區介面的網路遮罩，例如：255.255.255.0

IPv4 靜態路由子網路在這個表格裡可以填入子網路/網路遮罩、閘道器
子網路/網路遮罩為隔離區中某一段私有網段
閘道器為此私有網段需做靜態路由之閘道器地址
