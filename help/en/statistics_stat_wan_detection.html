WAN DetectionThis table shows the statistical data of WAN detection. These data can be used as reference to configure the parameters in [System/WAN Detection]. The WAN Detection table shows Number of Requests, Number of Replies, and Success Ratio % for each target IP listed in [System/WAN Detection/Target List]. Under circumstances of no disconnection, Success Ratio indicates the reliability of the IP, whether or not it is good for WAN Detection. IP addresses with high success ratio close to 100% are highly recommended.
Destination IP AddressThe IP addresses whose connections are to be detected.
Number of RequestsNumber of attempts sent out to the destination IP accumulated to date.
Number of RepliesNumber of replies received in this WAN link so far.
Success Ratio (%)The percentage obtained from the result of Number of Replies divided by Number of Requests. Higher ratio indicates the WAN link being more stable.
