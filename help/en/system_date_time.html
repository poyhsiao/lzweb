Date/Time

Date/Time Setting This table allows you to set up the Time Zone, Time Server, Date and Time of the system.
Time Zone Select a time zone from the drop-down menu.
Time Server XteraLink supports NTP time synchronization. Enter the IP address of a time server here or an FQDN number.
Date Enter a system date. e.g.
YYYY/MM/DD
Time Enter a system time. e.g.
HH:MM:SS
Set Date/Time Click it to immediately update system time. Other configurations will take effect until Apply button is clicked.
