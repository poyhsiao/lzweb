SNMP

SNMP SettingXteraLink supports SNMP v1 and SNMP v2 protocols.
This table allows you to configure Enable, Community, System Name, System Contact, and System Location for every rule.
EnableWhen unchecked (disabled), SNMP protocol is not enabled.
When checked (enabled), SNMP protocol is enabled.
CommunityEnter a corresponding Community that SNMP belongs.
System NameEnter a system name based on administration policy. This name can be can be defined by Administrator. e.g. AVT
System ContactEnter a contact name for the system based on administration policy. This name can be defined by Administrator. e.g. jackie
System LocationEnter a location where the system is, based on administration policy. This location name can be defined by Administrator. e.g. Taipei
