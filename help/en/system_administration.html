Administration

Account SettingThis table allows you to add, modify or delete XteraLink's user accounts, while monitoring every user account's status.
NameThe name of current user account.
GroupThe name of the group that current user account belongs to.
StatusThe status of current user account. e.g.
: Online
: Offline
AddAdd a new XteraLink user account.
EditEdit account settings.
DeleteDelete an existing XteraLink user account.
Sort by NameSort all accounts by user name.
Sort by GroupSort all accounts by user group.

MaintenanceThis table allows you to configure factory defaults and reboot system.
Factory defaultXteraLink will be restored back to factory default values after clicking "Factory default".
RebootSystem will be rebooted after clicking "Reboot".

Configuration FileThis table allows you to download or upload configuration file.
Save ConfigurationCurrent system configurations will be downloaded in a file after clicking "Save".
Restore ConfigurationSelect a previously saved configuration file and click "Restore", then the system will be restored back to previous settings saved in the uploaded file.

Firmware UpdateThis table allows you to perform firmware update or downgrade.
Current VersionThe version of current firmware used. e.g. Vx.x-Bxxxx-Mxxx-Px
V: firmware version
B: version number
M: Product model no.
V: Product hardware no.
Firmware Update
Input update keyTo update system to a different version (e.g. update file: Vx.x-Vx.x-Bxxxx-Mxxx-Px.update), an update key is required.
To update system within the same version (e.g. update file: Vx.x-Bxxxx-Mxxx-Px.update), an update key is not required.
System will be rebooted after clicking "Update".
Firmware DowngradeSelect a firmware of older version (e.g. Vx.x-Bx.x.x.x-Mxxxx-Px.downgrade), and the system will be rebooted after clicking "Downgrade".
