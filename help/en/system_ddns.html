DDNS

The DDNS (Dynamic Domain Name Server) of each WAN can be defined here. The DDNS must be registered at the provider's website prior to use.

WANSelect a WAN number from the drop-down menu to be configured.

DDNS Setting This table allows you to configure the DDNS setting of the selected WAN.
Enable When disabled, the selected WAN's DDNS will be unavailable.
When enabled, the selected WAN's DDNS will be activated based on the settings last applied.
Provider Select a provider's website where you want to apply for a DDNS from the drop-down menu. For example:
www.no-ip.com
Domain Name Enter the domain name obtained from the provider's website. e.g. xxx.com.tw
User Name Enter the domain name obtained from the provider's website. e.g. xxxx
Password Enter the password obtained from the provider's website. e.g. xxxx
